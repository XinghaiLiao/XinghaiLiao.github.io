Android中菜单栏的使用（上下文菜单）

2019-08-13

1.简介

上下文菜单提供了许多操作，这些操作影响 UI 中的特定项目或上下文框架。使用步骤如下：

  1. 通过调用 registerForContextMenu()，注册应与上下文菜单关联的 View 。

  2. 实现 onCreateContextMenu() 方法。

  3. 实现 onContextItemSelected()方法。

  4. 不需要使用的时候调用unregisterForContextMenu()方法注销对应的UI组件注册。
